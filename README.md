 # ¿Qué es la IA (Inteligencia Artificial)? 
La Inteligencia Artificial (IA) es una rama de la informática que se encarga de crear sistemas o programas capaces de realizar tareas que normalmente requieren inteligencia humana.

Estas tareas pueden incluir:

Aprender de la experiencia
Reconocer imágenes o voces
Tomar decisiones
Resolver problemas
Entender lenguaje humano

# Ejemplos de IA en la vida diaria:

Asistentes virtuales (como Siri o Alexa)
Recomendaciones de Netflix o YouTube
Autos autónomos
Chatbots

# ¿Qué es el Machine Learning? 🧠

El Machine Learning (aprendizaje automático) es una parte de la Inteligencia Artificial que permite a las computadoras aprender a partir de datos sin ser programadas explícitamente para cada tarea.
En lugar de escribir todas las reglas, se le dan muchos datos al sistema para que encuentre patrones y mejore con el tiempo.

# Ejemplo:
Un sistema puede aprender a reconocer si un correo es spam analizando miles de correos anteriores.
Tipos de Machine Learning (diferentes aprendizajes) 📊
1. Aprendizaje Supervisado
El modelo aprende usando datos etiquetados (con respuestas correctas).

# Ejemplo:
Fotos etiquetadas como "gato" o "perro"
El modelo aprende a distinguirlos.

# Usos:
Reconocimiento de imágenes
Predicción de precios
Clasificación de correos spam

# 2. Aprendizaje No Supervisado
El modelo recibe datos sin etiquetas y debe encontrar patrones por sí mismo.

# Ejemplo:
Agrupar clientes según su comportamiento de compra.

# Usos:
Segmentación de clientes
Detección de patrones
Agrupación de datos

# 3. Aprendizaje por Refuerzo
El modelo aprende probando acciones y recibiendo recompensas o castigos.

# Ejemplo:
Un programa que aprende a jugar ajedrez o videojuegos mejorando cada partida.

# Usos:
Robots
Juegos
Autos autónomos

# Inteligencia Artificial y Machine Learning

## ¿Qué es la IA?
La **Inteligencia Artificial (IA)** es una rama de la informática que crea sistemas capaces de realizar tareas que normalmente requieren inteligencia humana, como aprender, reconocer patrones, tomar decisiones o entender lenguaje.

## ¿Qué es el Machine Learning?
El **Machine Learning (Aprendizaje Automático)** es una parte de la IA que permite a las computadoras **aprender a partir de datos** sin ser programadas paso a paso. El sistema analiza información y encuentra patrones para mejorar sus resultados.

## Tipos de Machine Learning

### 1. Aprendizaje Supervisado
El modelo aprende con **datos etiquetados** (con respuestas correctas).

**Ejemplo:** clasificar correos como spam o no spam.

### 2. Aprendizaje No Supervisado
El modelo trabaja con **datos sin etiquetas** y encuentra patrones o grupos por sí mismo.

**Ejemplo:** agrupar clientes según su comportamiento de compra.

### 3. Aprendizaje por Refuerzo
El modelo aprende mediante **prueba y error**, recibiendo recompensas o castigos según sus acciones.

**Ejemplo:** programas que aprenden a jugar videojuegos o ajedrez.

# Data Frame vs Data Set

## ¿Qué es un Data Set?
Un **Data Set (conjunto de datos)** es una **colección organizada de datos** relacionados entre sí.

Características:
- Puede estar en una tabla, archivo o base de datos.
- Contiene **filas (registros)** y **columnas (variables)**.
- Se usa para almacenar información para análisis.

Ejemplo:

| Nombre | Edad | Ciudad |
|------|------|------|
| Ana | 25 | Córdoba |
| Luis | 30 | Buenos Aires |

Esto completo es un **dataset**.

---

## ¿Qué es un Data Frame?
Un **Data Frame** es una **estructura de datos en forma de tabla** usada en programación para trabajar con datasets.

Se utiliza mucho en lenguajes como:
- **Python** (con la librería Pandas)
- **R**

Características:
- Tiene filas y columnas
- Cada columna puede tener un tipo de dato diferente
- Permite filtrar, ordenar y analizar datos fácilmente

Ejemplo conceptual de DataFrame:

| ID | Producto | Precio |
|----|----------|--------|
| 1 | Laptop | 900 |
| 2 | Mouse | 20 |

---

## Diferencia principal

| Data Set | Data Frame |
|---------|------------|
| Es el **conjunto de datos** | Es una **estructura para manipular esos datos** |
| Puede existir en archivos o bases de datos | Se usa dentro de programas (Python, R) |
| Concepto general de datos | Herramienta para trabajar con esos datos |

**En resumen:**
- **Dataset = los datos**
- **DataFrame = la estructura para trabajar con esos datos**
